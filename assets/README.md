# Assets

Place these files here before building:

| File | Size | Purpose |
|---|---|---|
| `icon.png` | 1024×1024 | App icon |
| `splash.png` | 1242×2436 | Splash/loading screen |
| `adaptive-icon.png` | 1024×1024 | Android adaptive icon foreground |
| `favicon.png` | 32×32 | Browser tab favicon |
| `icon-192.png` | 192×192 | PWA icon (Android home screen) |
| `icon-512.png` | 512×512 | PWA icon (splash + high-DPI) |

## Quick placeholder icons

You can generate placeholder icons using any image editor, or use a tool like:
- https://www.appicon.co (generates all sizes from one 1024×1024 image)
- https://favicon.io (for favicon.png)

## Suggested design

- Background: `#0f172a` (deep navy)
- Foreground: `#818cf8` (indigo) with "A" lettermark or a simple geometric shape
- The name "Aria" in the splash screen is rendered in code, so the image can be abstract
